# Tidy Haven

A polished, responsive brochure site for a residential professional-organizing business.

## Local preview

```bash
cd /Users/nimbus/tidy-haven-organizing
python3 -m http.server 8080
```

Then visit http://localhost:8080.

## Deployment

The repository is configured for Vercel as a zero-build static site. Import it in Vercel or deploy with the Vercel CLI after authenticating.

## Before launch

- Replace `hello@tidyhaven.co` with the actual business email.
- Replace stock Unsplash photography with licensed brand photography.
- Add the real Instagram URL.
- Connect the consultation CTA to a scheduling tool or lead form.
